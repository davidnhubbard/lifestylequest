# AGENTS.md

## Project goal
Build a simple lead-magnet web app called "Design Your Dream Life in Ecuador."

## Working style
- Work in small verified steps.
- Do not redesign the product without being asked.
- Keep code readable and minimal.
- Use Next.js App Router and Tailwind.
- Prefer simple components over abstraction.
- After each step, report:
  - objective
  - files changed
  - result
  - next step
- Never say "done" without verifying.

## MVP priorities
1. Clear landing page
2. Smooth question flow
3. Good results summary
4. Lead capture
5. Email delivery
6. Deployability

## Constraints
- No overengineering
- No unnecessary libraries
- Make Vercel deployment easy
- Use Supabase for storage
- Use Resend for emails
